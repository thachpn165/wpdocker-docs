# wpdocker wp plugin auto-update

Enable plugin auto update

## Usage

```bash
wpdocker wp plugin auto-update DOMAIN ACTION
```

## Arguments

#### *DOMAIN*

Domain name of the site to enable plugin auto update

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *ACTION*

Action to perform (enable, disable)

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
