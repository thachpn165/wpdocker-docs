# wpdocker backup restore database

Restore database only

## Usage

```bash
wpdocker backup restore database DOMAIN DB_BACKUP_FILE
```

## Arguments

#### *DOMAIN*

The domain will perform database recovery

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *DB_BACKUP_FILE*

The path to the database backup file

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
