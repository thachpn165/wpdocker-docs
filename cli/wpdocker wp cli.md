# wpdocker wp cli

Run WP-CLI commands inside container. Usage wpdocker wp cli wpdocker.dev "\<command\>"

## Usage

```bash
wpdocker wp cli DOMAIN COMMAND
```

## Arguments

#### *DOMAIN*

Domain name of the site to run WP-CLI commands

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *COMMAND*

WP-CLI command to run

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
