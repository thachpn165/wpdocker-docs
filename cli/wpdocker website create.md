# wpdocker website create

Create a new WordPress site

## Usage

```bash
wpdocker website create DOMAIN PHP [AUTO_GENERATE]
```

## Arguments

#### *DOMAIN*

Domain name for the new site

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *PHP*

PHP version for the new site

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *AUTO_GENERATE*

Auto-generate secure passwords for the site

| Attributes      | &nbsp;
|-----------------|-------------
| Default Value:  | true
