# wpdocker backup restore file

Restore file only

## Usage

```bash
wpdocker backup restore file DOMAIN CODE_BACKUP_FILE
```

## Arguments

#### *DOMAIN*

The domain will perform file recovery

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *CODE_BACKUP_FILE*

The path to the backup file for the site

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
