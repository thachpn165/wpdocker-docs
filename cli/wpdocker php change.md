# wpdocker php change

Change PHP version for a site

## Usage

```bash
wpdocker php change DOMAIN VERSION
```

## Arguments

#### *DOMAIN*

Domain name of the site to change PHP version

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *VERSION*

New PHP version for the site

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
