# wpdocker wp security login

Protect wp-login.php for a site

## Usage

```bash
wpdocker wp security login DOMAIN ACTION
```

## Arguments

#### *DOMAIN*

Domain name of the site to protect wp-login.php

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *ACTION*

Action to perform (enable, disable)

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
